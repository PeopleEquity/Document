使用备注#

  1. 以 markdown 格式编写内容
  2. 每个文件夹下，可以重复创建子文件夹
  3. 图片等资源统一放在 ./static/img/下 （注: 放在对应项目文件夹中，避免混淆）
  4. 使用示例，参考ESChain/Demo.md
  5. 零散的博客放在与docs并列的blog文件夹下

如有使用问题，随时联系 老张。
